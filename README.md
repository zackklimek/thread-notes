# \# thread-notes

Vue3, TypeScript, Google Firebase, Langchain

Project setup:

1. Clone into this repository
2. `npm install`
3. Set up Firebase for project: Authentication, Firestore, Storage
4. Create firebaseConfig.ts in '/'
5. Add `VITE_OPENAI_API_KEY` to .env
6. `npm run dev`
